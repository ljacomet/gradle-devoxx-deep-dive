== What is Gradle?

Gradle is a build and automation tool.

Gradle can automate the building, testing, publishing, deployment and more of software packages
or other types of projects such as generated static websites, generated documentation or indeed anything else.

* JVM based
* Implemented in Java
* 100% Free Open Source - Apache Standard License 2.0

=== Gradle Inc.

The company behind Gradle.

* Build Happiness

* Providing Gradle Build Scans and Gradle Enterprise
